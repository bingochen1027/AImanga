# Step 8B｜NeoTV 价格、案例中心与投放 Skill 包改造说明

## 1. 本次改造目标

在 Step 8A 已经完成「NamiStory 分阶段产线 + Neo / Neowow Skill 引导」的基础上，本轮继续改造外围商业与内容页面：

- `pricing.html`：改成 NeoTV 风格的低门槛价格方案。
- `templates.html`：从题材模板扩展为「题材模板 + 技能工作流模板」。
- `showcase.html`：参考 NeoTV 作品墙，升级案例中心。
- `launch-kit.html`：把投放包升级为「投放技能包」。
- `checkout.html`：同步 NeoTV 风格价格方案，补充更多购买方案。
- `account.html`：同步会员、积分和项目包状态。

> 说明：NeoTV / NeoDomain 公开页面无法稳定抓取完整价格表，本轮采用「NeoTV 式价格结构」进行原型设计：低门槛会员、积分充值、团队会员、项目包、投放包、企业定制。

---

## 2. Pricing 改造重点

### 2.1 原问题

上一版价格体系偏 B 端项目制，价格梯度较高：

- 免费试用
- 订阅会员 ¥99-199/月
- 积分包 ¥29-299
- 项目包 ¥2,999 起
- 企业版定制

这和 NeoTV / NeoDomain 这类面向创作者的内容产品相比，入门门槛偏高，不利于中国 ToC 流量从首页、模板和案例进入后快速转化。

### 2.2 新价格体系

新版改为「低门槛开通 + 按需充值 + 团队升级 + 项目增购」：

| 类型 | 名称 | 示例价格 | 作用 |
|---|---|---:|---|
| 免费 | 免费试用 | ¥0 | 新用户验证第一集草稿 |
| 会员 | 创作者会员 | ¥29/月起 | 个人创作者低门槛持续使用 |
| 会员 | 专业会员 | ¥99/月 | 高频创作、系列化、批量生成 |
| 团队 | 团队会员 | ¥299/月起 | 工作室 / MCN / 小团队协作 |
| 积分 | 小额积分包 | ¥9.9 / 100积分 | 轻量试错 |
| 积分 | 标准积分包 | ¥49 / 600积分 | 日常生成 |
| 积分 | 大额积分包 | ¥299 / 5000积分 | 批量视频和投放测试 |
| 项目 | 首集试制包 | ¥199起 | 1集试制和基础投放素材 |
| 项目 | 10集连载包 | ¥999起 | 题材测试和资产复用 |
| 投放 | 投放素材包 | ¥399起 | 标题、封面、3秒钩子、多版本测试 |
| 企业 | 企业版 | 定制 | 长期合作、私有资产库和服务保障 |

## 3. 案例中心改造重点

### 3.1 参考 NeoTV 的展示形式

根据 NeoTV 页面截图，本轮把案例中心从普通案例列表升级为：

1. 顶部活动轮播区域
2. 最近画布 / 最近项目入口
3. 深色作品墙
4. 题材分类 Tabs
5. 作品卡片瀑布流
6. 案例过程拆解

### 3.2 案例卡片方向

案例中心加入参考 NeoTV 页面中出现的作品类型与名称：

- 元岛
- 万物重生
- 地火
- EPHEMERAL LIGHT
- 断线
- 刀剑如梦
- 痞颜高校
- PACKAGE

这些案例用于展示题材覆盖，而不是替代真实客户案例。后续如果需要正式上线，需要替换为授权作品、真实数据和可公开素材。

## 5. Launch Kit 改造重点

投放包升级为五个技能：

| 技能 | 输出 |
|---|---|
| 标题爆点技能 | 10 个标题，按悬念、爽点、反转、情绪分类 |
| 封面文案技能 | 5 张封面方向和人物特写建议 |
| 3 秒钩子技能 | 3 个开场版本：冲突、反转、情绪 |
| 评论引导技能 | 角色口吻回复、追更话术和小程序引导 |
| 复盘优化技能 | 根据点击率、完播和转化收益给出下一轮建议 |

购买后输出清单包括：

- 10 个标题
- 5 张封面方向
- 3 条 3 秒钩子脚本
- 多版本测试矩阵
- 数据复盘表
- 下一轮优化建议
