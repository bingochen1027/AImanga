# 付费路径说明 Step 6

## 主路径

```text
价格页 / 个人中心 / 抖音投放包
        ↓
订单确认页 checkout.html
        ↓
支付成功页 payment-success.html
        ↓
个人中心 account.html
        ↓
继续创作 / 生成投放包 / 下载授权 / 申请发票
```

## 入口对应关系

| 入口 | 链接 | 说明 |
|---|---|---|
| 免费试用 | `login.html` | 新用户领取试用额度 |
| 订阅会员 | `checkout.html?plan=creator` | 开通创作者专业版 |
| 积分充值包 | `checkout.html?plan=credits` | 购买额外积分 |
| 项目包 / 投放包 | `checkout.html?plan=project` | 购买项目权益或投放素材权益 |
| 企业版 | `checkout.html?plan=enterprise` | 提交企业咨询和定制方案 |

## 支付成功后的权益承接

| 购买类型 | 支付成功后显示 | 个人中心承接模块 |
|---|---|---|
| 订阅会员 | 会员权益和月度积分已更新 | 当前套餐、积分钱包、订单 / 发票、商用授权 |
| 积分充值包 | 积分余额已更新 | 我的钱包 / 积分、订单 / 发票 |
| 项目包 / 投放包 | 项目包权益已写入项目空间 | 我的项目、已购项目包 / 投放包、商用授权 |
| 企业版 | 企业咨询记录已生成 | 订单 / 发票、企业合作记录、后续客户成功跟进 |
