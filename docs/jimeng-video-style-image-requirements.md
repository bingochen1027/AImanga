# 即梦生成图要求：视频风格卡片

用途：用于 `creator-studio.html` 的“视频设定 / 视频风格”卡片缩略图。

## 统一要求

- 数量：9 张，每个视频风格 1 张。
- 画幅：横版 16:9。
- 建议尺寸：1600 x 900，最低不要低于 1280 x 720。
- 画面：主体清晰，适合网页卡片缩略图，中间区域保留核心视觉信息。
- 禁止：不要文字、不要字幕、不要 logo、不要水印、不要边框、不要二维码。
- 人物：可以出现通用原创人物，但不要使用明星脸、影视角色、动漫 IP 或真实品牌。
- 输出格式：优先 JPG，也可以 WEBP。文件名需要按下表保持一致。
- 放置目录：`frontend/assets/images/video-styles/`

## 通用负面提示词

文字，字幕，logo，水印，二维码，品牌标识，低清晰度，模糊，过曝，严重畸变，人物多手指，脸部崩坏，真实明星脸，已知影视角色，已知动漫 IP，血腥，恐怖过度，裸露。
