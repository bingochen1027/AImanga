# 界面修改说明

## 最方便的方式

打开 `frontend/edit.html`，选择要改的页面。页面进入编辑模式后，点击蓝色虚线区域即可临时修改文字、按钮或图片路径。

临时修改只保存在当前浏览器中。页面右下角点击“导出修改”会复制一段 JSON，你可以交给 Codex 写入 `frontend/assets/site-overrides.js`，这样修改就会固定在项目里。

## 固定修改的位置

- 常见文案、按钮、案例标题、案例图片、价格页文案：`frontend/assets/site-overrides.js`
- 首页结构：`frontend/index.html`
- 案例中心结构：`frontend/showcase.html`
- 价格页结构：`frontend/pricing.html`
- 登录页：`frontend/login.html`
- 项目向导：`frontend/wizard.html`
- 引导式工作台步骤：`frontend/assets/creator-studio.js`
- 全站颜色、间距、卡片、按钮样式：`frontend/assets/styles.css`
- 后台 Portal 文案和数据：`portal/index.html`、`portal/assets/portal.js`

## 图片替换

建议把新图片放到：

```text
frontend/assets/images/story-cases/
```

然后在编辑模式里把图片路径改成类似：

```text
assets/images/story-cases/your-new-image.png
```

## 预览方式

在项目根目录启动静态预览服务后打开：

```text
http://localhost:4173/frontend/edit.html
```

如果只是临时看页面，也可以打开：

```text
frontend/index.html
```

但为了让前台和后台本地状态更稳定，建议用本地服务预览。
